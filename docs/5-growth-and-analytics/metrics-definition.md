# Metrics Definition Doc

## Progression System
- XP points — users earn XP and level up.
- No composite "Pull Score." Keep it simple with XP.
- Badges and milestones should be subtle and professional (no childish gamification).

## Identity Tiers
Beginner → Conversationalist → Connector → Room Leader. Progression based on XP + lessons completed + application accuracy.

## Scoring Categories (per roleplay attempt)
1. Application of Principle (primary)
2. Warmth
3. Clarity
4. Confidence
5. Humor
6. Listening

## Engagement Metrics to Track
- Lessons completed
- Roleplay sessions completed
- Streak (consecutive days)
- Notes logged
- Reflection prompts filled in

## Notifications (Phase 2)
- Push notification 24 hours after last app usage (Duolingo pattern: if they had free time yesterday at 2pm, they likely have free time today at 2pm).
